# Azure Functions Scripts

PowerShell scripts for managing Azure Function Apps using Azure CLI.

## Prerequisites

- Azure CLI (https://docs.microsoft.com/en-us/cli/azure/install-azure-cli)
- Active Azure subscription and logged-in CLI session (`az login`)
- An existing Azure Storage Account (required by Function Apps)

## Available Scripts

| Script                           | Description                                                                                |
| -------------------------------- | ------------------------------------------------------------------------------------------ |
| `az-cli-create-function-app.ps1` | Create an Azure Function App with configurable runtime, OS type, and optional hosting plan |
